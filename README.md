# Micro-PSH-Gate-Control
Control algorithm (SCL) for an automated gate system of a 10 kW micro hydropower plant in Uzbekistan. Includes adaptive solar-based control and jamming protection.
## Repository Structure

* **`Source_Code/`**: **[FB_ValveMotorControl_V10.scl](./FB_ValveMotorControl_V10.scl)**: Automated control logic (SCL) for the gate system, including jamming protection and solar-based adaptive control.
* **`Technical_Data/`**: [Technical Specifications and Engineering Data.pdf](./Technical%20Specifications%20and%20Engineering%20Data.pdf) — contains calculations, hydraulic parameters, and equipment specifications for the 10 kW plant.
